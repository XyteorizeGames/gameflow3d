# The Gameflow3D Game Framework

Gameflow3D is a python game engine/framework for building cross platform games (Android/IOS/Windows/Mac/Linux)

# Installation

Download from the releases and click ```gf3dInstall-1.0.0-win.exe``` or ```gf3dInstall-1.0.0-macnlin.sh```
